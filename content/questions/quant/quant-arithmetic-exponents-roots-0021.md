## Question

What is \(\sqrt{50} + \sqrt{18}\) in simplest form?

## Explanation

You can only add radicals once the parts **under** the root match, so simplify each one by pulling
out perfect squares:

\[\sqrt{50} = \sqrt{25 \cdot 2} = 5\sqrt{2}, \qquad \sqrt{18} = \sqrt{9 \cdot 2} = 3\sqrt{2}\]

Now both terms are multiples of \(\sqrt{2}\), so add the coefficients:

\[5\sqrt{2} + 3\sqrt{2} = 8\sqrt{2}\]

**The traps:**

- **A (\(2\sqrt{17}\))** comes from illegally merging the roots: \(\sqrt{50} + \sqrt{18} \neq \sqrt{68}\).
  Square roots do not add that way.
- **B (\(15\sqrt{2}\))** multiplies the coefficients (\(5 \times 3\)) instead of adding them.

## Hints

- Simplify each radical first by factoring out the largest perfect square.
- You can add radicals only when the number under the root is the same — then add the coefficients.
