## Question

How many integer values of \(x\) satisfy the inequality \(|x - 3| < 5\)?

## Explanation

An absolute-value inequality \(|A| < b\) (with \(b > 0\)) unfolds into a **double inequality**
\(-b < A < b\). Here:

\[|x - 3| < 5 \implies -5 < x - 3 < 5\]

Add 3 to all three parts:

\[-2 < x < 8\]

The inequality is **strict**, so \(x = -2\) and \(x = 8\) are excluded. The integers strictly
between \(-2\) and \(8\) are:

\[-1,\ 0,\ 1,\ 2,\ 3,\ 4,\ 5,\ 6,\ 7\]

That's **9** integers.

**Watch the endpoints:** because the sign is \(<\) and not \(\le\), don't count \(-2\) or \(8\).
Miscounting the boundaries is the most common error here.

## Hints

- Rewrite \(|x-3| < 5\) as \(-5 < x - 3 < 5\), then isolate \(x\).
- The inequality is strict, so the endpoints are not included — count only the integers strictly between them.
