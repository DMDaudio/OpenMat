## Question

Every item in the store that is on sale is from last season's collection. None of the items in
the store's front window display is from last season's collection.

If the statements above are true, which of the following must also be true?

## Explanation

Chain the two facts about the same categories:

- Every **on-sale** item is a **last-season** item. (on-sale → last-season)
- **No** front-window item is a last-season item. (front-window → not last-season)

If something is on sale, it must be last season's; but nothing in the front window is last
season's. So nothing in the front window can be on sale.

Therefore **(A): none of the front-window items is on sale.** This *must* be true. **Correct.**

Why the others fail:

- **(B)** and **(E)** reverse the logic — the premises say nothing forces non-sale or sale items *into* the window.
- **(C)** reverses the first premise: on-sale items are all last-season, but not every last-season item need be on sale.
- **(D)** directly contradicts the second premise (nothing in the window is last-season).

## Hints

- Write each statement as an arrow (on-sale → last-season; front-window → not last-season) and look for a chain.
- A valid inference *must* follow; eliminate any choice that merely *could* be true or that reverses a premise.
