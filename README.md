# 60-Day GMAT Prep Program

A two-month, daily interactive GMAT study program covering **Quantitative**, **Verbal**,
**Integrated Reasoning**, and **Analytical Writing Assessment (AWA)**.

## Contents

| File | Purpose |
|------|---------|
| [`curriculum.html`](curriculum.html) | Interactive curriculum hub — phase overview, filterable day-by-day plan, and local progress tracking. **Start here.** |
| [`curriculum.md`](curriculum.md) | Canonical machine-readable plan. The daily runs consult this to build each app. |
| `day-NN-<section>-<topic>.html` | One interactive study app per day (Days 2–60), created on its scheduled day. |

## How it works

- **Day 1** produced the curriculum (`curriculum.md` + `curriculum.html`).
- **Days 2–60** each add one self-contained interactive HTML study app containing a concept
  explanation with examples, 5–10 practice problems with instant feedback and hints, live score
  tracking, and a key-takeaways summary. Each app links back to the curriculum and forward to the
  next day.
