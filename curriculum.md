# 60-Day GMAT Prep Curriculum

> **Canonical reference for the daily study-app program.** Every subsequent daily run
> (Days 2–60) must consult this file to determine that day's section, topic, learning
> objectives, practice focus, and difficulty level, then build the interactive HTML study app.

- **Program length:** 60 days
- **Day 1:** Curriculum planning (this document + `curriculum.html`)
- **Days 2–60:** One interactive HTML study app per day (59 apps total)
- **Sections covered:** Quantitative (Q), Verbal (V), Integrated Reasoning (IR), Analytical Writing Assessment (AWA)
- **Start date of program:** Day 1 = 2026-07-07

---

## How to use this file on Days 2–60

1. Determine **today's day number** (Day 1 = 2026-07-07; Day N = 2026-07-07 + (N−1) days).
   If unsure, count the existing `day-*.html` files already committed and take the next one.
2. Look up that day number in the **Day-by-Day Plan** table below.
3. Build an interactive HTML study app that includes:
   - Concept explanation with worked examples
   - 5–10 practice problems with immediate feedback
   - Score tracking and hints for incorrect answers
   - A summary of key takeaways
4. Save it using the naming convention: **`day-NN-<section>-<topic-slug>.html`**
   (e.g. `day-02-quantitative-arithmetic.html`, `day-05-verbal-critical-reasoning.html`).
5. In each app include a **link back to `curriculum.html`** and a **forward link to tomorrow's file**.

### File naming convention

`day-NN-<section>-<topic-slug>.html`

- `NN` = zero-padded day number (`02`…`60`)
- `<section>` = `quantitative` | `verbal` | `integrated-reasoning` | `awa` | `mixed`
- `<topic-slug>` = short kebab-case topic (see "File" column in the plan)

## Difficulty progression (4 phases)

| Phase | Days | Theme | Difficulty |
|-------|------|-------|------------|
| **1 — Foundations** | 2–16 | Rebuild fundamentals, learn each question format from scratch | Foundational |
| **2 — Core skill building** | 17–36 | Master the highest-frequency topics and techniques | Intermediate |
| **3 — Advanced application** | 37–52 | Hard problems, traps, mixed-topic reasoning, speed | Advanced |
| **4 — Test readiness** | 53–60 | Timed section simulations, error-log targeting, test-day strategy | Mixed / exam-like |

Diagnostic checkpoints at **Day 16** and **Day 36**; full-length simulation at **Day 59**.

---

## Section coverage summary

- **Quantitative (28 days):** Arithmetic, percents/ratios, number properties, algebra (linear, quadratics, exponents, inequalities), word problems (rates, work, mixtures, sets), geometry (lines/triangles/quads/circles/coordinate), statistics, probability, combinatorics, Data Sufficiency, and timed mixed sets.
- **Verbal (18 days):** Sentence Correction (grammar, agreement, modifiers, parallelism, tense, comparisons/idioms), Critical Reasoning (structure, assumptions, strengthen/weaken, inference, evaluate/flaw/boldface, paradox), Reading Comprehension (main idea, detail/inference, structure, dense passages/tone), plus timed mixed sets.
- **Integrated Reasoning (7 days):** Graphics Interpretation, Table Analysis, Two-Part Analysis, Multi-Source Reasoning, mixed and full-section simulation.
- **AWA (3 days):** Analysis of an Argument foundations, flaw analysis + templates, timed essay with self-scoring.
- **Mixed / test-readiness (5 days):** Checkpoints, section simulations, error-log review, full-length test, final strategy.

## Notes for consistency across daily apps

- **Design:** Self-contained single HTML file (inline CSS + JS, no external dependencies). Light/dark friendly, mobile responsive.
- **Interactivity:** Multiple-choice or entry practice with instant right/wrong feedback, a per-question hint revealed after a wrong attempt, a running score counter, and a reset button.
- **Structure of each app:** (1) header with day/section/topic + link to `curriculum.html`; (2) concept explanation with worked examples; (3) 5–10 practice problems with feedback/hints; (4) key-takeaways summary; (5) footer with prev/next navigation.
- **Progress persistence:** Use `localStorage` for per-app score where helpful.
- **Tone:** Encouraging, exam-focused, concise explanations.

> **Scope note:** This plan covers the four classic GMAT sections named in the program brief
> (Quant, Verbal, IR, AWA). The current *GMAT Focus Edition* replaces AWA/Sentence Correction
> with an expanded "Data Insights" section; the IR and grammar work here transfers directly, so
> learners on either format are served. Adjust emphasis if targeting Focus Edition specifically.
